# Task Files

Task files are the small, executable prompts for new agent windows.

Use them with:
- `.agents/skills/sanmarino-step-runner/SKILL.md`
- `docs/ai/PROJECT_INDEX.md`
- `docs/ai/CONTEXT_BRIEF.md`
- `docs/ai/BUSINESS_RULES.md`

Rules:
- one task file per agent window
- no broad implementation outside the task
- task files do not override accepted docs
- if a task conflicts with canonical docs, stop and report
